# Arduino_RC_car
RC car made using Arduino controller

Power: 4x 18650 batteries + LM2596 step-down converter
Logic: Arduino R4 Uno Wifi
Motor Control: 2x L9110H H-Bridge controller
Movement: 4x DC motors 6V
